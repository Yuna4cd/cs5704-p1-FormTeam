# cs5704-p1-FormTeam

Team Member: Jiyuan Zhang (jy1229)

Overview:

FormTeam is a web-based system designed to address the issue of uneven strength between the competing teams. Event organizers usually need to spend a lot of time manually assessing the skill levels of the contestants, but this method yields inaccurate results. FormTeam helps them skip this step and provides a balanced team composition. This automated process can be divided into teams, record the competition results, and adjust the skill levels of the participants based on the results, thereby ensuring the accuracy of the team formation balance in the future.
