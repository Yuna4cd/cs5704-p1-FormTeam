# cs5704-p1-FormTeam

Team Member: Jiyuan Zhang (jy1229)

Overview:

To solve this problem, the web-based system proposed in this article, FormTeam, can address the issue by balancing the strength differences between teams. This system adjusts the skill level in real time based on the participants' historical records. Furthermore, this system uses a client-server architecture to reduce the burden on the user's device, centrally manage important data, and efficiently generate team combinations.

In summary, the FormTeam system provides a solution that can solve the problem of unbalanced teams. Event organizers can use this system to save significant time while maintaining the participant's experience. In the future, we'll adapt the system to multiple devices and run user testing again to ensure the product is usable.
